# Paper Trading Simulator (Python)

A basic paper trading simulator built in Python to model
buy/sell decisions, transaction costs, and portfolio value
over time.

## Features
- Tracks cash and stock positions
- Simulates transaction costs
- Logs trade history
- Generates portfolio equity curve

## How to Run
1. Install Python 3.13+
2. Install matplotlib: `pip install matplotlib`
3. Run `simulator.py` in Python or Colab

## Motivation
Built to understand market mechanics and portfolio behavior
through first-principles simulation.
